# Technical Blog Exploration

I wanted to try every popular static site generator that could host a technical blog (code + math).

## Gatsby

```bash
cd gatsby
gatsby develop
```

## Hugo

```bash
cd hugo
hugo serve -t LoveIt --disableFastRender
```

## Next

```bash
cd next
pnpm dev
```

## Vitepress

```bash
cd vitepress
pnpm docs:dev
```
